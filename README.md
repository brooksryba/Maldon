# Maldon
Python based command-line tool for managing .flake8 settings

## About
Maldon is a utility for managing flake8 configurations across projects. This tool provides human-readable descriptions for each of the PEP8 style rules, and allows you to manage presets to use when creating new projects or improving old ones.
